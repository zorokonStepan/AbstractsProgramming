```
    docker image - готовое к запуску приложение
    docker image – это шаблон для работающего контейнера.
    docker image состоит из слоев.
    
    docker container - запущенное приложение на базе docker image
    
    из docker container нельзя изменить docker image(READ ONLY)
    
    Изменения в запущенных контейнерах можно сохранять и тегировать как новые образы
```
