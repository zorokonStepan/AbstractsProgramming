# Виды шаблонов проектирования
```
    1. Порождающие
    2. Структурные
    3. Поведенческие
```

## Порождающие шаблоны проектирования
```
    Порождающие шаблоны описывают создание (instantiate) объекта или группы связанных объектов.
    
    1. Простая фабрика
    2. Фабричный метод
    3. Абстрактная фабрика
    4. Строитель
    5. Прототип
    6. Одиночка
```
