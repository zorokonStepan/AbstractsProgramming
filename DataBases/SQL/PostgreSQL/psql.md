```
    \? - Для получения краткой справки по всем сервисным командам
         Многие такие команды начинаются с символов «\d»
    
    \dt - просмотреть список всех таблиц и представлений (views), созданных в той базе данных, к которой сейчас подключены
    
    \d students - структура какой-либо конкретной таблицы базы данных, например, students
    
    \h - Для получения списка всех SQL-команд
    
    \h CREATE TABLE - Для вывода описания конкретной SQL-команды
```
