```
    1. CREATE TABLE - Создание таблицы
    2. DROP TABLE - Удаление таблицы
    3. ALTER TABLE - Изменение таблицы
     
    4. Оператор INSERT (INSERT TO) - Вставка данных в таблицы
    5. Оператор UPDATE - Изменение данных в таблице
    6. Оператор DELETE - Удаление данных из таблиц        
    
    7. SELECT - выборка данных
    8. FROM - откуда, имя таблицы
    9. LIMIT - Ограничение количества извлекаемых строк
    10. AS - Создание псевдонима  
    
    11. WHERE - Фильтрация строк, с которыми будут выполняться действия, 
               Используется не только в SELECT, но и других операторах SQL
                 
    12. Операторы сравнения   
        =, !=, <>, >, >=, <, <=
        IN, BETWEEN, LIKE
        
        8.1. BETWEEN - WHERE amount BETWEEN 5 AND 14; Включительно!
        8.2. IN - Выборка по вхождению в кортеж: WHERE author IN ('Булгаков М.А.', 'Достоевский Ф.М.');
        
    13. LIKE - Выборка по подстроке: WHERE title LIKE '%_Б%';   
        
        Специальные символы в шаблонах LIKE
            % - любое количество символов (включая 0)
            _ - ровно один символ 
            
    14. Логические операции
            AND, OR, NOT       
            
    15. Оператор IS NULL       
        
    16. ORDER BY - Сортировка данных в SELECT. Возможна сортировка по одному или нескольким столбцам
        
        Порядок сортировки
            ASC (ascending) – сортировка по возрастанию
            DESC (descending) – сортировка по убыванию
            
            
    
    17. DISTINCT - уникальные значения
    18. ALL - противоположность DISTINCT. Если не заданы ни ALL ни DISTINCT, по умолчанию используется ALL.
    19. GROUP BY - Группировка по одному или нескольким столбцам
    20. HAVING - В запросах с групповыми функциями вместо WHERE используется ключевое слово HAVING,
                 которое размещается после оператора GROUP BY.
                  
    21. Агрегатные(групповые) функции:
        COUNT, 
        SUM, 
        AVG, 
        MAX, 
        MIN    
        
        COUNT(*) — подсчитывает  все записи, относящиеся к группе, в том числе и со значением NULL;
        COUNT(имя_столбца) — возвращает количество записей конкретного столбца (только NOT NULL), относящихся к группе.           
     
     
    100. Математические функции:
                                CEILING(x)
                                ROUND(x, k)
                                FLOOR(x)
                                POWER(x, y)
                                SQRT(x)
                                DEGREES(x)
                                RADIANS(x)
                                ABS(x)
                                PI()
                                    e.t.c.
        
    101. /*  */ - многострочный комментарий   
     
    102. IF - IF(логическое_выражение, выражение_1, выражение_2) в MySQL
```

```
    Совместное использование ключевых слов в SELECT
    WHERE, ORDER BY, LIMIT
    
    Группировка и другие возможности SELECT
    Фильтрация: WHERE
    Сортировка: ORDER BY
    Ограничение количества строк: LIMIT
```

# Examples:
```
    1. CREATE TABLE - создать таблицу
    
        CREATE TABLE name_table(
            field TYPE_FIELD,
            ...,
            ...
        );
```
```
    2. INSERT TO - вставка записи в таблицу
    
    INSERT INTO таблица(поле1, поле2) 
        VALUES (значение1, значение2);
        
    или
    
    INSERT таблица(поле1, поле2) 
        VALUES (значение1, значение2);    
```
```
    3. SELECT - выборка данных
    
    SELECT * FROM book;
    
    SELECT title, amount FROM book;
    
    SELECT title AS Название, amount FROM book;
    
    SELECT price * amount AS total FROM book;
```
```
    12. DISTINCT - уникальные значения
    
        SELECT DISTINCT author FROM book;
```
```
    13. GROUP BY - сгруппировать по столбцам
    
    С помощью GROUP BY можно выбрать уникальные элементы столбца, по которому осуществляется группировка.
     Результат будет точно такой же как при использовании DISTINCT
     
        SELECT DISTINCT author FROM book;
    
        SELECT author FROM book GROUP BY author;
```
