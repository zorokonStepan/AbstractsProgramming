```
    __copy__(self) - Определяет поведение copy.copy() для экземпляра вашего класса. copy.copy() 
                     возвращает поверхностную копию вашего объекта — это означает, что хоть сам объект и создан заново, 
                     все его данные ссылаются на данные оригинального объекта. И при изменении данных нового объекта, 
                     изменения будут происходить и в оригинальном.

    __deepcopy__(self, memodict={}) - Определяет поведение copy.deepcopy() для экземпляров 
                                      вашего класса. copy.deepcopy() возвращает глубокую копию вашего объекта — 
                                      копируются и объект и его данные. memodict это кэш предыдущих скопированных 
                                      объектов, он предназначен для оптимизации копирования и предотвращения 
                                      бесконечной рекурсии, когда копируются рекурсивные структуры данных. 
                                      Когда вы хотите полностью скопировать какой-нибудь конкретный атрибут, 
                                      вызовите на нём copy.deepcopy() с первым параметром memodict.
```
