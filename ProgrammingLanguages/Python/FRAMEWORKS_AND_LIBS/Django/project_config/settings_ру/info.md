```
    INSTALLED_APPS = [
        'django.contrib.admin',
        'django.contrib.auth',
        'django.contrib.contenttypes',
        'django.contrib.sessions',
        'django.contrib.messages',
        'django.contrib.staticfiles',
    ]
    
    Список, хранящийся в переменной INSTALLED_APPS, перечисляет все приложения,
    зарегистрированные в проекте и участвующие в его работе. 
    
    Изначально в этом списке присутствуют только стандартные приложения, входящие в состав Django и
    реализующие различные встроенные подсистемы фреймворка. 
    
    Так, приложение
    django.contrib.auth реализует подсистему разграничения доступа, а приложение
    django.contrib.sessions — подсистему, обслуживающую серверные сессии.
```
