```
    1. Создание через конструктор класса + метод save()
    Cозданная таким образом запись модели не сохраняется в базе данных, а существует только в оперативной памяти. 
    Чтобы сохранить ее, достаточно вызвать у нее метод save().
    
    ===================================================================================================================
    
    from bboard, modeIs import Bboard
    
    b = Bboard(title='Дача', content='Общество "Двухэтажники",  price=500000)
    b.save()
     
    b.pk -> 1
    если существует pk, значит запись создана    
    
    ===================================================================================================================
    
    b2 = Bboard()
    b2.title = "sss"
    b2.content = "ddd"
    b2.save()
    
    b.pk -> 2
    
    2. Метод create() диспетчера записей создает новую запись модели, принимая в качестве набора именованных параметров 
    значения ее полей, сразу же сохраняет ее и возвращает в качестве результата.
```
