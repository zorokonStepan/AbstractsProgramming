    @pytest.fixture(scope="module")
     
    default: scope="function"
     
    scope='function'
        Запускается один раз для каждой тестовой функции. 
        Часть настройки выполняется перед каждым тестом с использованием fixture. 
        Этап разборки выполняется после каждого теста с использованием fixture. 
        Это область по умолчанию, используемая, если параметр области не указан.
  
    scope='class'
        Выполняется один раз для каждого тестового класса, независимо от количества тестовых методов в классе.
    
    scope='module'
        Запускается один раз для каждого модуля, независимо от того, сколько тестовых функций, методов или других 
        fixture в модуле используют его.
    
    scope='package'
        Запускается один раз для каждого пакета или каталога тестирования, независимо от того, сколько тестовых 
        функций, методов или других fixture в пакете используют его.
    
    scope='session'
        Запускается один раз за сеанс. Все методы и функции тестирования, использующие fixture области сеанса, 
        используют один вызов установки и демонтажа.
