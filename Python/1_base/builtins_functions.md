### Получение справки
```
    dir() - пытается вернуть список допустимых атрибутов для данного объекта. 
            Если аргумент не указан, он возвращает список имен в текущей локальной области.
    
    help() -> keywords - вывод всех зарезервированных слов  
```

```
    chr(65) --> "A"
    ord("A") --> 65
```
