# commands
```
    manage.ру runserver
        Команда runserver запускает отладочный веб-сервер написанный на Python
        
    manage.ру startapp app
        Команда startapp утилиты manage.py запускает создание нового "пустого" приложения, 
        имя которого указано после этой команды.
```
