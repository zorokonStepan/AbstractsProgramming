### Атрибуты class`a Meta
```
    атрибуты класса Meta, задают параметры самой модели
    
    1. verbose_name_plural — название модели во множественном числе;
    2. verbose_name — название модели в единственном числе. 
    Эти названия будут выводиться на экран админки;
    3. ordering — последовательность полей, по которым по умолчанию будет выполняться сортировка записей
```

```
    class Meta:
        verbose_name_plural = 'Объявления'
        verbose_name = 'Объявление'
        ordering = ['-published']
```
