# order_by()
```
    Метод order_by() диспетчера записей сортирует записи по значению поля, имя которого указано в параметре, 
    и сразу же возвращает набор записей, получившийся в результате сортировки.
    
    for b in Bb.objects.order_by('title'):
        print(b.pk, ': ', b.title)
         
    ===================================================================================================================
    
    for bb in Bb.objects.order_by('-published')
    
    Чтобы отсортировать объявления по убыванию даты их публикации, в вызове метода order_by() диспетчера записей 
    предварили имя поля published символом "минус"
```
