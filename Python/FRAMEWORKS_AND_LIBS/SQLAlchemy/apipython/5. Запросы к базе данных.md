    SQLAlchemy предоставляет различные методы для выполнения запросов к базе данных. 
    Наиболее распространенными являются методы query и filter.

    Метод query используется для выполнения выборки записей из таблицы. 
    Например, для получения всех записей таблицы User:

        users = session.query(User).all()

    Метод filter используется для фильтрации записей таблицы. 
    Например, для получения всех пользователей с именем “John”:

        users = session.query(User).filter(User.name == 'John').all()

    Также можно выполнить несколько фильтров:

        users = session.query(User).filter(User.name == 'John', User.age <= 30).all()

    Для выполнения более сложных запросов может быть полезен метод join. 
    Он позволяет объединить две или более таблицы по определенному критерию.
    Пример использования:

        result = session.query(User, Address)
                        .join(Address, User.id == Address.user_id)
                        .filter(Address.city == 'Moscow')
                        .all()

    Результатом будет список кортежей, содержащих объекты User и Address, у которых адрес находится в городе “Moscow”.

    Aggregate функции, такие как count, sum или avg, могут быть использованы для получения суммарной информации из 
    таблицы. Например, чтобы узнать, сколько пользователей в таблице User:

        num_of_users = session.query(User).count()

    Все эти методы могут быть использованы вместе, чтобы создавать более сложные запросы к базе данных.
