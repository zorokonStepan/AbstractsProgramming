# zip()
```
    в базовом виде функция zip принимает одну или большее количество последовательностей в качестве аргументов и 
    возвращает серию кортежей, объединяющих в пары параллельные элементы из указанных последовательностей. 
    
    L1 = [1, 2, 3, 4]
    L2 = [5, 6, 7]
    
    for i in zip(L1, L2):
        print(i)
    
    >>>  (1, 5)  <class 'tuple'>
    >>>  (2, 6)  <class 'tuple'>
    >>>  (3, 7)  <class 'tuple'>
    
    
    T1, Т2, ТЗ = (1,2,3), (4,5,6), (7,8,9)
    list(zip(T1, Т2, ТЗ))
    [(1, 4, 7), (2, 5, 8), (3, 6, 9)]
```
