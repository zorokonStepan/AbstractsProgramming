    Приоритет:
    
    1. class __get__ + __set__
    2. obj __dict__
    3. class __get__ (в том числе @classmethod, @staticmethod, def func, lambda)
    
    Для экземпляра класса пп. 1 и 3 статичны, они определены в классе. В экземпляре класса
    на ходу можно поменять пп. 2 и 3, а п.1 на ходу не изменить.
