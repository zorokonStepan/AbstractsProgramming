```
    GraphQL — это язык запросов, используемый клиентскими приложениями для работы с данными. 
    
    GraphQL — это система, независимая от источников данных.
    
    Схема GraphQL - это то, что позволяет организовывать создание, чтение, обновление и удаление данных в вашем 
    приложении (то есть — перед нами четыре базовые функции, используемые при работе с хранилищами данных, которые 
    обычно обозначают акронимом CRUD — create, read, update, delete).
```

### Запросы GraphQL
```
    Запросы GraphQL — это сущности, представляющие собой запрос к серверу на получение неких данных.
    
    {
      flavors {
        id
        name
        description
      }
    }
       
    flavors, id, name, description - поля запроса в схеме
```

#### Аргументы запроса
```
    {
      flavors(id: "1") {
        id
        name
        description
      }
    }
    
        или динамически
    
    query getFlavor($id: ID) {
      flavors(id: $id) {
        id
        name
        description
      }
    }
    
    query - тип запроса
    getFlavor - имя запроса, имя выбирается произвольным образом!!!
```
